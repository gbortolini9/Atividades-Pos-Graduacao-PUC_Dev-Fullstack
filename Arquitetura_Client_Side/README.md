# Arquitetura Client Side

Este diretório contém as atividades desenvolvidas para a disciplina **Arquitetura Client-Side** do curso de Pós-Graduação em Desenvolvimento Full Stack da PUC.

O objetivo das atividades é explorar conceitos fundamentais da arquitetura de aplicações web modernas, incluindo **Single Page Applications (SPA)**, **componentização**, **micro frontends** e o uso de **frameworks baseados em React**.

---

# Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- React
- Next.js

---

# Estrutura das atividades

## Atividade 1 — Micro Frontends e App Shell

Demonstração de uma arquitetura baseada em **micro frontends**, utilizando um **container principal (App Shell)** responsável por carregar diferentes módulos da aplicação.

Conceitos abordados:

- Arquitetura App Shell
- Micro Frontends
- SPA (Single Page Application)
- Roteamento client-side
- Separação de responsabilidades
