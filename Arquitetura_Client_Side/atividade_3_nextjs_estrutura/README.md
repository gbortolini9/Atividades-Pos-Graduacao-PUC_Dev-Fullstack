# Atividade 3 - Estrutura com Next.js

Esta atividade demonstra a utilização do framework Next.js para estruturar uma aplicação React de forma organizada e escalável.

## Conceitos aplicados

- Framework para aplicações React
- Roteamento por páginas
- Componentização
- Separação de responsabilidades
- Reutilização de componentes

## Estrutura

- `pages` → páginas da aplicação
- `components` → componentes reutilizáveis
- `styles` → estilos globais

## Tecnologias utilizadas

- Next.js
- React
- JavaScript
- CSS
