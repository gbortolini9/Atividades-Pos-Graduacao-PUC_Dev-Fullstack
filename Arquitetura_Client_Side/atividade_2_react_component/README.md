# Atividade 2 - Componentes React

Esta atividade demonstra a criação de componentes visuais utilizando React e JSX.

## Conceitos aplicados

- Componentes React
- JSX
- Props
- Renderização dinâmica

## Descrição

Foi criado um componente chamado `Saudacao` responsável por renderizar uma mensagem na tela.

Esse componente recebe dados externos através de **props**, permitindo reutilização do mesmo componente para diferentes usuários.

## Tecnologias

- React
- JSX
- HTML
- CSS
