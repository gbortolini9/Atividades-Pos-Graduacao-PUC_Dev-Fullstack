# Atividade 5 - Busca de CEP com API

Esta atividade tem como objetivo praticar o consumo de API em uma aplicação web, utilizando HTML, CSS e JavaScript.

## Funcionalidades
- Buscar endereço a partir de um CEP
- Validar formato do CEP
- Exibir logradouro, bairro, cidade e estado
- Tratar erros de CEP inválido ou não encontrado

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- API ViaCEP
