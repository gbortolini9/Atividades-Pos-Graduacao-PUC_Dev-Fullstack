# Atividade 2 - Lista de Tarefas

Esta atividade tem como objetivo praticar conceitos de Programação Web por meio de uma lista de tarefas interativa.

## Funcionalidades
- Adicionar tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Persistir dados no navegador com localStorage

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
