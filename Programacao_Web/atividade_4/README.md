# Atividade 4 - Mini Carrinho de Compras

Esta atividade consiste em um mini sistema de carrinho de compras desenvolvido com HTML, CSS e JavaScript.

## Funcionalidades

- Listagem de produtos
- Adição de produtos ao carrinho
- Remoção de produtos
- Cálculo automático do valor total
- Persistência do carrinho usando localStorage

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
