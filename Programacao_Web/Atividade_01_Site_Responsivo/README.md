# Atividade 1 - Cadastro de Usuários

Esta atividade tem como objetivo praticar conceitos básicos de Programação Web com HTML, CSS e JavaScript.

## Funcionalidades
- Cadastrar usuários
- Validar campos obrigatórios
- Listar usuários na tela
- Remover usuários cadastrados

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
