# Exercício 01 - SOAP + XML

Este exercício tem como objetivo demonstrar o funcionamento básico de um Web Service no padrão **SOAP**, utilizando mensagens em **XML**.

A atividade apresenta exemplos de requisição e resposta para operações simples de uma calculadora, além de um arquivo **WSDL** ilustrando como o serviço pode ser descrito.

## Objetivos

- Compreender a estrutura de uma mensagem SOAP;
- Identificar o uso do XML na troca de dados;
- Entender o papel do WSDL na descrição de serviços;
- Visualizar exemplos práticos de requisição e resposta.

## Conceitos abordados

- Web Services
- SOAP
- XML
- WSDL
- Comunicação cliente-servidor via HTTP

## Estrutura do exercício

- `exemplos/`: contém exemplos de requisições e respostas SOAP em XML;
- `wsdl/`: contém um exemplo de arquivo WSDL para descrição do serviço.

## Exemplo de operação

O serviço simula operações matemáticas simples, como:

- soma
- multiplicação

## Resultado esperado

Ao analisar os arquivos, é possível entender como um cliente envia uma requisição SOAP em XML e como o servidor pode responder no mesmo formato.
